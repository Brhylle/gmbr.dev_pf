- [Bun](https://bun.sh) installed on your system

### Setup

1. Install dependencies:
```bash
bun install
```

2. Run development server:
```bash
bun run dev
```

3. Build for production:
```bash
bun run build
```

4. Preview production build:
```bash
bun run preview
```
